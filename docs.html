GET Отправка кода подтверждения

/telegram/login/phone/1?num={phone}
Отправляет код подтверждения на указанный номер телефона.
📌 Параметры запроса
num
string
Номер телефона в международном формате (например: +79257025543)
required
📤 Пример ответа: 
{
  "success": true,
  "data": {
    "message": "Code sent",
    "temp_id": "a1b2c3d4e5f6..."
  }
}
📝 Пример запроса (cURL): 
curl "http://localhost:8080/telegram/login/phone/1?num=%2B79257025543"

GET Подтверждение кода

/telegram/login/phone/2?code={code}&temp_id={temp_id}
Подтверждает код из SMS и возвращает данные сессии.
📌 Параметры запроса
code
string
Код из SMS
required
temp_id
string
Временный ID из предыдущего запроса
required
📤 Пример ответа: 
{
  "success": true,
  "data": {
    "session_data": "base64_encoded_session_string",
    "message": "Login successful"
  }
}
📝 Пример запроса (cURL): 
curl "http://localhost:8080/telegram/login/phone/2?code=12345&temp_id=a1b2c3d4e5f6"

POST Информация о себе

/telegram/get_me
Возвращает информацию о текущем авторизованном пользователе.
📌 Тело запроса
session.data
string
Данные сессии в base64
required
📤 Пример ответа: 
{
  "success": true,
  "data": {
    "ID": 123456789,
    "FirstName": "Влад",
    "LastName": "Гроза",
    "Username": "Endlad7373",
    "Phone": "79257025543"
  }
}
📝 Пример запроса (cURL): 
curl -X POST http://localhost:8080/telegram/get_me \
  -H "Content-Type: application/json" \
  -d '{"session": {"data": "base64_session_string"}}'

POST Информация о пользователе

/telegram/get_user_info
Возвращает информацию о пользователе по username, включая фото профиля.
📌 Тело запроса
session.data
string
Данные сессии в base64
required
username
string
Username пользователя (с @ или без)
required
📤 Пример ответа: 
{
  "success": true,
  "data": {
    "id": 123456789,
    "first_name": "Pavel",
    "last_name": "Durov",
    "username": "durov",
    "is_bot": false,
    "is_verified": true,
    "is_premium": false,
    "photo": {
      "has_photo": true,
      "photo_base64": "/9j/4AAQSkZJRg...",
      "photo_size": 12345
    }
  }
}

POST Отправка сообщения

/telegram/send_message
Отправляет текстовое сообщение пользователю или в чат.
📌 Тело запроса
session.data
string
Данные сессии в base64
required
peer
string
Username получателя (без @)
required
message
string
Текст сообщения
required
📤 Пример ответа: 
{
  "success": true
}

POST Получение сообщений

/telegram/get_messages
Получает историю сообщений с указанным пользователем.
📌 Тело запроса
session.data
string
Данные сессии в base64
required
peer
string
Username пользователя (без @)
required
limit
int
Количество сообщений (по умолчанию 50)
optional
📤 Пример ответа: 
{
  "success": true,
  "data": [
    {
      "id": 123,
      "text": "Hello!",
      "date": 1700000000,
      "out": false
    }
  ]
}

required
limit
int

optional
